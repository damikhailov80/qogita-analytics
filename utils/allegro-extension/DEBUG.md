# Как дебажить расширение

## 1. Логи Background Script (Service Worker)
1. Откройте `chrome://extensions/`
2. Найдите "Allegro Price Extractor"
3. Нажмите на ссылку "service worker" (или "Проверить представления: service worker")
4. Откроется DevTools для background script
5. Смотрите вкладку Console - там будут все логи с префиксом `[Allegro Extension]`

## 2. Логи Content Script (на странице localhost:3000)
1. Откройте страницу `http://localhost:3000/products/allegro`
2. Откройте DevTools (F12)
3. Во вкладке Console смотрите логи при наведении на ссылки

## 3. Логи на странице Allegro
1. Когда откроется вкладка с Allegro
2. Откройте DevTools (F12) на этой вкладке
3. Смотрите Console - там будут логи извлечения цены

## 4. Проверка разрешений
В `chrome://extensions/` убедитесь что расширение имеет:
- Доступ к localhost:3000
- Доступ к allegro.pl
- Разрешение на clipboard

## 5. Перезагрузка расширения
После изменения кода:
1. Откройте `chrome://extensions/`
2. Нажмите кнопку "Обновить" (круговая стрелка) на карточке расширения
3. Перезагрузите страницу localhost:3000

## Что проверить если не работает:

### Проблема: Вкладка открывается, но цена не копируется
- Проверьте логи в DevTools открытой вкладки Allegro
- Возможно селектор `.opbox-listing li p[tabindex="0"]` не находит элементы
- Попробуйте увеличить задержку в background.js (сейчас 2000ms)

### Проблема: Ничего не происходит при наведении
- Проверьте Console на странице localhost:3000
- Должно быть сообщение "Content script loaded"
- При наведении должно быть "Allegro link hovered"

### Проблема: Ошибки разрешений
- Убедитесь что в manifest.json есть все нужные permissions
- Перезагрузите расширение
