# Allegro Price Extractor

Chrome расширение для автоматического извлечения цен с Allegro.

## Установка

### Шаг 1: Откройте страницу расширений Chrome
1. Откройте браузер Google Chrome
2. В адресной строке введите `chrome://extensions/` и нажмите Enter
3. Или через меню: Три точки → Расширения → Управление расширениями

### Шаг 2: Включите режим разработчика
1. В правом верхнем углу страницы найдите переключатель "Режим разработчика" (Developer mode)
2. Включите его (переключатель должен стать синим)

### Шаг 3: Загрузите расширение
1. После включения режима разработчика появятся новые кнопки
2. Нажмите "Загрузить распакованное расширение" (Load unpacked)
3. В открывшемся окне выбора папки перейдите к вашему проекту
4. Выберите папку `utils/allegro-extension`
5. Нажмите "Выбрать папку" (Select Folder)

### Шаг 4: Проверьте установку
1. Расширение "Allegro Price Extractor" должно появиться в списке
2. Убедитесь что переключатель расширения включен (синий)
3. Проверьте что нет ошибок (красных значков)

### Обновление расширения
После изменения кода расширения:
1. Откройте `chrome://extensions/`
2. Найдите "Allegro Price Extractor"
3. Нажмите кнопку с круговой стрелкой (Обновить)
4. Перезагрузите страницу `http://localhost:3000/products/allegro`

## Использование

### На странице Products Allegro
1. Откройте страницу `http://localhost:3000/products/allegro`
2. Зажмите клавишу Shift
3. Наведите курсор на ссылку, ведущую на Allegro
4. Расширение автоматически:
   - Откроет ссылку в соседней вкладке
   - Извлечет первую цену со страницы
   - Скопирует ее в буфер обмена
   - Закроет вкладку
   - Найдет кнопку с соответствующим GTIN
   - Кликнет на неё и заполнит input ценой

### На странице Sellers
1. Откройте страницу `http://localhost:3000/sellers/{sellerId}`
2. Зажмите клавишу Shift
3. Наведите курсор на ссылку, ведущую на Allegro
4. Расширение выполнит те же действия

## Как это работает

- `manifest.json` - конфигурация расширения
- `content.js` - отслеживает наведение на ссылки Allegro на localhost:3000
- `background.js` - открывает вкладки и выполняет скрипт извлечения цены
